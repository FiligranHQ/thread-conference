This page is a visual catalogue — every component, color rule and mosaic style used across the THREAD site, isolated so the design and dev team can copy behaviors 1:1. For exact tokens (hex, spacing, type scale) see the companion Design System document.

01 — Color usage rules

Where each color is allowed to live

The palette is intentionally wide. To keep the site calm on a dark background, each color has a defined role — some are UI-safe (buttons, links), others are accent-only (tags, top-lines, mosaics).

Cool set — UI & interaction

✓Blue, Cyan, Lime — used for buttons, links, the gradient, focus states, and the primary interactive accent (cyan) across the whole site.
✓The signature gradient (blue → cyan → lime) is the only gradient allowed. It never includes warm colors.
✕Never use Coral, Magenta or Pink in a gradient, a primary button, or as body-text color.

Warm set — accents & categorization

✓Coral, Magenta, Pink, Purple — reserved for tags, badges, top-lines on cards, and speaker/voice category markers.
✓Each of the 6 experience zones and 4 speaker categories gets one distinct accent color for quick visual recognition.
✕Never combine coral + orange-leaning tones together or with pink in a single gradient (avoids an "Instagram" look).
02 — Buttons

Buttons — cool palette only

Primary actions always use the cool set. No coral/orange button ever exists in the system.

03 — Tags & badges

Full expressive palette

Tags can use any color in the palette (including coral/pink) since they carry small amounts of color in a controlled shape — never large surfaces.

04 — Zone / experience cards

Colored top-line, outlined body

Each of the 6 experience zones gets one accent — applied only as a 4px top-line + a subtle border tint. The card body stays dark/outlined, never a full color fill.

05 — Generative mosaic art

Grainy pixel-block texture

The mosaic is the strongest brand signature — it echoes "noise becomes signal". Always grainy, always scattered (never a solid tiled grid), used as background texture behind hero sections, register bands, venue maps and section dividers. Never as a flat color fill.

Low density — subtle background texture (hero mask)

Medium density — venue map / section accent
